Notes

  • † — The character code shown is decimal rather than hexidecimal.
  • Webpages: HTML "&#x<unicode>;" (e.g. "a&xF02B;b" → ab)
  • Microsoft Word: Type the unicode hexidecimal code followed by [alt]X (e.g. F02B+[alt]+X) or use the insert/Special Character... menu instead. It does not matter if you type the unicode number in upper or lower case.
  • Underdots are not used as diacritical marks in Ancient Greek, these are used to highlight uncertain letters in a reading of a manuscript.

Over-lines

The unicode character 0305 provides an overline. If the overline character fails to overlay the target character, this may be due to the font you are using, try Arial Unicode MS, Lucinda Sans Unicode or Gentium.

  • On a webpage this can be formatted as "&#x0305;". For example "a&#x0305;b" → a̅b
  • In Windows using Microsoft Word this would be 0305[alt]x.
