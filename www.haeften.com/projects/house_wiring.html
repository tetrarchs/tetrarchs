Volterra Database

A voluntary project for the History department at University College London to put a database of ancient roman law texts on the internet (originally in MS Access but UCL IT would not allow a direct link for a database).
I developed this solution using PERL scripts and embedding various JavaScript tweaks to the web pages. StyleSheets are also used to keep the pages consistent. The challenge was to design the system so that historians could update the text datafiles, even to the extent of adding new database tables, and the PERL scripts would not have to be changed.

Flipper Game

Quick game that "flips" a tile on a board and the surrounding tiles. The objective is to flip all the tiles to "red".
All written in JavaScript with two small gif images (10x10 pixels) being used. Uses frames with the main frame being generated from the first.

Prime Calculator

Calculates the prime factors for any number (type in the number and then click anywhere else on the page to generate the results). May take a couple of seconds the first time you try a large number (seven digits plus).
Written in JavaScript this page can be saved off-line and still works okay.

Ascii Character Table

Displays a nice table of characters (system font) against character code.
The source html is short as a simple JavaScript loop is used to generate the table. Characters generated are from ascii codes 32 to 4096.
