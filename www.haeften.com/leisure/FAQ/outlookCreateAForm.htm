FAQ: Creating an Outlook Form with Buttons

1. Open Forms Editor

The first step is to find the Outlook forms edit in the toolbar. Select the standard mail template and use this to create another form (once happy with your new form you can use Formsadmin.exe from microsoft to replace the standard mail form, Google for some guidance as you may need to search an replace the Outlook version number in the resulting registry file).

2. Create Buttons

You can move around all the normal part of the mail form and you can use the controls toolbox to introduce drop lists and buttons.

I used buttons to introduce quick controls for standard email types used in my company. In particular the convention is to prefix the subject line with INFO, REQ (request), Action etc. so the recipient can quickly see what to deal with first. The suffix [eom] is also used for "pager" emails where the subject line is all that is needed (e.g. "Drinks tonight on at 8pm not 7.30 [eom]".

3. Create Script

Once the buttons are set up neatly on your form, the tricky part is the associated code. Use the "view code" button to open an edit panel for scripts to be used with the form. Below are my example scripts for buttons I named InfoButton, ReqButton, UrgentButton, PerButton and EomButton. As you create the scripts you can run the form to check there are no bugs introduced. Use the object browser as part of the visual basic editor (alt F11 in Outlook) to find the properties available for the mail item object. 

Example Code

Sub InfoButton_Click()
	item.subject="INFO " + item.subject
	item.categories="Information"
end sub
Sub ReqButton_Click()
	item.subject="REQ " + item.subject
	item.categories="Request"
end sub
Sub UrgentButton_Click()
	item.subject="Urgent " + item.subject
	item.categories="Urgent"
	item.importance=2	'0=low 2=high
end sub
Sub PerButton_Click()
	item.categories="Personal"
	item.sensitivity=2	'1=personal
end sub
Sub EomButton_Click()
	item.subject=item.subject+" [eom]"
	item.categories="Brief Note"
	item.body=item.body+"Key: [eom] means 'end of message'"
end sub
4. Use the form

Save the form under the personal forms library. You can either choose the form when you want it from the menu or use the Formsadmin tool from Microsoft to replace the standard form with yours.
