Mobile phone unlocking or flashing cable

Sony Ericsson K750 models (SE K750)

Description

For sale is a cable used for unlocking K750 / W800 type mobile phones. This is the type that does not need power but just connects your computer to the phone and the USB port should be recognized if you have the right sort of software.

I used the cable to unlock my own K750i (using software from totalmultiserver.com) and then reprogrammed it with a W800i firmware image that I downloaded using eMule (as the only hardware difference between the K750 and W800 was styling). I shopped around for the cheapest site to give me an unlocking code (in fact I bought the unlocking code in an eBay auction which went via a site in the USA) and suggest you do the same. Considering that you might be charged £30 or so to unlock each phone it can be a pretty good saving.

Using this cable takes a bit of research, so please don't bid on this unless you are prepared to take this on as a minor technical project or have unlocked phones yourself before.
